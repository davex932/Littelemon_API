# Littlelemon_API

## 1-Description du projet

  une API REST construitr avec django rest_framwork permettant de gérer le menu, la mise en panier des article, la validation des commandes et même gestion des utilisateurs.

## 2-Fontionnalites

-operations CRUD sur les items du menu.

-possibilites de créer,supprimer et afficher deux types d'utilisateurs:le manager et le livreur.

-possibilites de mettre, supprimer et afficher les items dans le panier.

-possibilites de creer une commande.
